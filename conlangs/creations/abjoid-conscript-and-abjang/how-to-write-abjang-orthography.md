# How to write AbjAng Orthography

AbjAng is usable in a variety of ways, as is any script and any orthography.

I have derived and constructed my own orthography for modern English, which I've currently dubbed _NewEng_, for want of a better name. It aims to include pronunciation, etymological indication and grammatical markings; the latter two aspects are in current English orthography, but they are the reason that the spelling system does not relate to the pronunciation. Additionally, the spelling system of the current English orthography is dialectical in nature, in that it does contain enough flexibility to allow for a broad variety of accents to use it without too much trouble. _NewEng_ aims to maintain this dialectality too.

AbjAng orthography is built upon the NewEng orthography, and as such, you can mentally process NewEng directly into AbjAng. An AbjAng font would need a few glyph modifications to allow for the additional letters of the NewEng orthography, and indeed my first Abjoid Latinate font does include glyphs for most (if not all) NewEng letters.

### Choose a keyboard layout

* You can type directly in Latin using your regular keyboard. This will not give you NewEng letters or diacritics!
* You can type directly in NewEng using my NewEng keyboard layout. _At this time, I think my AbjAng NewEng font does not yet contain diacritics, which are fundamental to the current version of NewEng._
* You can also type in base Abjoid using my Abjoid keyboard layout. This is more raw, as you will type the consonants and vowels with the left hand, and the consonant modifiers with the right hand. _This keyboard layout also does not include any keys for NewEng diacritics (yet)._
