# Chatting about language

If you want to learn a language in a natural and organic way, you should do it just like how babies and children do it!

First step: learn how to ask "how do i say this in Vietnamese?"&#x20;

Then learn basic grammatical terms. Don't worry about confusing people — most of these terms in Vietnamese are very simple, self-explanatory terminology.

{% hint style="info" %}
#### Grammatical word names are simple

noun — danh từ — name word&#x20;

adjective — tính từ — character word&#x20;

verb — động từ — motion word&#x20;

adverb — trạng từ — condition/state word&#x20;

preposition — giới từ — connect word&#x20;
{% endhint %}
